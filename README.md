# Lost & Found App

This project is a simple Streamlit web application designed to manage and explore information related to lost and found items. It demonstrates the use of Python for building an interactive web interface and handling structured data.

## Features
- Interactive web app built using Streamlit
- Displays and processes lost and found item data
- Simple and user-friendly interface
- Uses a CSV dataset for data storage and retrieval

## Tech Stack
- Python
- Streamlit
- Pandas

## Project Structure
- app.py — Main Streamlit application file
- lost_found_data.csv — Dataset containing lost and found item information
- requirements.txt — Required Python dependencies
- .devcontainer — Development container configuration
- Lost and Found.zip — Archived project files

- Worked with structured data using Pandas
